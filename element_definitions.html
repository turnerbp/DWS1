HTML Element Definitions

HTML
Abbreviation for HyperText Markup Language.
H1
This is the Main Heading.
H2
This is a sub heading
H3
This is another sub heading
P
means Paragraph
ul
unordered list
ol
Ordered list
li
list item
span
used to group in-line html
em
emphasis
strong
shows strong emphasis for its content
b
displays elements in bold
i
italic element
small
element use for small print
cite
Defines an in-line citation or reference to another source.
q
An in-line quote. Used for small quotations.
dfn
Definition term.
abbr
Defines an element that is a shortened word or phrase.
time
It is used to make dates and times rich.
code
Defines code (such as computer code).
figure
Element to mark up and annotate figures or illustrations
figcaption
Element to mark up and annotate figures or illustrations
var
Variable. Defines a computer variable for example.
samp
Defines sample output, from a computer program for example.
kbd
Used to define text that should be typed in by the user.
sub
subscript
sup
superscript
mark
Highlights text in a document
base
Defines the base location for links on a page
A
Anchor. Primarily used as a hypertext link.
href
Used to link webpages or email
